# Tab Operations Skill

This skill automatically triggers the tab-management agent when you work on tabs.

## When This Triggers

- Tab creation and lifecycle
- Tab template management
- Tab groups and layouts
- Output buffering
- Tab reordering
- Auto-shutdown
- Split view layouts

## Quick Start

The tab-management agent has comprehensive knowledge of:
- Tab lifecycle management
- Tab template system
- Split view layouts (horizontal, vertical, grid)
- Output buffer management
- Tab groups
- Auto-shutdown configuration
