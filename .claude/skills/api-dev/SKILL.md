# API Development Skill

This skill automatically triggers the api-database agent when you work on APIs or database changes.

## When This Triggers

- Creating new API endpoints
- Modifying database schema
- Generating and applying migrations
- Adding Zod validation schemas
- Implementing authentication
- Permission checking
- Query optimization
- Error handling

The api-database agent has comprehensive knowledge of:
- Next.js 15 App Router patterns (async params)
- Zod validation schemas
- Drizzle ORM query patterns
- PostgreSQL timestamp compatibility
- Error handling and response formatting
- Migration workflow
