## Video Content (10 seconds)

**0-2 sec**: User clicks into the Name field, types: "feature-api-v2"

**2-4 sec**: User clicks Branch dropdown. List opens showing branches:
- main
- develop
- feature/auth
- feature/dashboard
- feature/new-api

User starts typing "feat" - list filters to only feature branches.

**4-5 sec**: User selects "feature/new-api" from filtered list.

**5-6 sec**: Template already selected (default). User clicks "Create" button.

**6-8 sec**: Modal closes. Progress indicator appears in sidebar next to new workspace. Status shows yellow "Starting" with subtle pulse animation.

**8-10 sec**: Status transitions from yellow "Starting" to green "Running". Text overlay: "Deploy in seconds."
