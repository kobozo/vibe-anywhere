- Professional dark theme

## Video Content (10 seconds)

**0-3 sec**: Quick montage begins. Fast cuts between:
- Different workspaces being selected (cursor flies)
- Terminal commands executing
- Split views activating and deactivating
- Git diffs appearing

Each cut is 0.5 seconds. Energy is high.

**3-5 sec**: More montage:
- Code scrolling
- Tests passing
- Files being committed
- New workspace deploying

**5-7 sec**: Zoom out effect. Camera pulls back to show the entire dashboard. All workspaces visible, multiple showing activity.

**7-8 sec**: Final establishing shot. Everything running in parallel. Status indicators pulsing subtly.

**8-9 sec**: Text overlay fades in: "Vibe Anywhere"

**9-10 sec**: Tagline appears below: "Your coding sessions. Always running."
Optional: Website URL fades in below tagline

### Motion & Effects
- Fast cuts (0.5s each)
- Zoom out effect
- Activity in multiple workspaces
- Text fade-in sequence
- Final logo/tagline treatment
- Subtle pulse on status indicators
