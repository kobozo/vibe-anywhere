## Video Content (10 seconds)

**0-2 sec**: User clicks on "Node.js 22" card. Card highlights with border and checkmark appears.

**2-3 sec**: User clicks on "Claude Code" card. Same selection effect.

**3-4 sec**: User clicks "Continue" or "Create Template" button.

**4-6 sec**: View transitions to a staging terminal. Terminal shows installation in progress:
```
Installing Node.js 22...
✓ Node.js 22.11.0 installed

Installing Claude Code...
Downloading claude-code@latest...
✓ Claude Code installed
```

Packages download and install with progress indicators.

**6-8 sec**: Installation continues with more output scrolling.

**8-9 sec**: Final success message:
```
✓ All tech stacks installed successfully
Template ready for use!
```

**9-10 sec**: Text overlay: "Pre-configured. Ready to code."

### Motion & Effects
- Card selection animation (scale + border)
- Checkmark appear animation
- Terminal scroll animation
- Progress indicators
- Success checkmarks appearing
- Text fade-in
