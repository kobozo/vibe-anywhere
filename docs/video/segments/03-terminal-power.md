## First Frame (Screenshot Description)

Full-screen terminal view occupying the main content area. Terminal characteristics:

- **Font**: JetBrains Mono, clean and readable
- **Background**: Dark theme (#1a1a2e or similar)
- **Content**: A test suite output mid-run
  - Green checkmarks (✓) for passing tests
  - Test names with timing (e.g., "✓ should authenticate user (45ms)")

  - Progress indicator visible

The terminal feels professional and polished, like a native desktop application.
