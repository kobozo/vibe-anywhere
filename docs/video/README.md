# Vibe Anywhere - Promotional Video Assets

This folder contains descriptions and specifications for creating promotional videos for Vibe Anywhere.

## Overview

The video content is designed as a series of **10-second segments** that can be:
- Used individually for social media
- Combined into a full 100-second (1:40) promotional video
- Mixed and matched for different campaign needs

### Key Messages
- "Your AI coding sessions. Always running."
- "Parallel isolated sessions. Zero conflicts."
- "Close your laptop. Come back anytime."
- "Pre-configured. Ready to code."

### Visual Style
- **Theme**: Dark mode
- **Font**: JetBrains Mono (terminals), Inter/SF Pro (UI)
- **Primary Colors**:
  - Success/Running: `#10b981` (green)
  - Error/Stopped: `#ef4444` (red)
  - Warning/Starting: `#f59e0b` (yellow)
  - Primary accent: `#3b82f6` (blue)

## Production Notes

Each segment file includes:
- Detailed first frame description
- Second-by-second video breakdown
- Motion and effects notes
- Production checklist

## Getting Started

1. Read [VIDEO-SEGMENTS.md](VIDEO-SEGMENTS.md) for the full picture
2. Review individual segments for detailed specs
3. Capture/create screenshots matching first frame descriptions
4. Record/animate video content per specifications
5. Use production checklists to verify completeness
